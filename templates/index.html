Elected Prosecutor Database

Welcome to the Elected Prosecutors Database powered by Color Of Change. We hope this will become a useful tool for locating and evaluating potential targets in the landscape of prosecutor reform. The database is comprised of publicly available information about districts, elections and candidates for the more than 2,400 elected prosecutors in the country. It contains over 100,000 cells of manually researched data compiled in just a few short months

This database is currently in its Minimum Viable Product stage, meaning this is simply a barebones version used to gather information about how users would interact with the information. We need your help to shape the future of the database and to report any issue areas so that we can develop a fully functional tool over the next several years. Please contact Color Of Change at prosecutordb@ColorOfChange.org to set up a 1:1 User Session and make frequent use of the Report an Issue function above.




How to access the Database


Register with Us

Once you have been approved, you can access the report builder to help your organization identify which candidates, elections, and districts show promise for change.

Get Approval

Once you register, your info is sent to our the Prosecutor Database team. Our team will grant your access to the database and send your login instructions. Please note that approval can take up to 48 hours.

Start Querying

Once you have been approved you can access ready made reports on our user database, as well as a report builder to help your organization identify which candidates, elections, and district show promise for change.

Help Us Improve our Database!

Please bear in mind this database in its current form is a minimum viable product. That's tech jargon for "a version of a new product which allows a team to collect the maximum amount of validated learning about users with the least amount of development." That's where you come in: we need help to make this a fully functional tool. Please use the "Report an Issue" function to tell us about any usability issues or data errors you notice.
